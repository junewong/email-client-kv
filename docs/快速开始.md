# 快速开始指南

## 开发者快速开始

### 1. 克隆项目

```bash
git clone https://github.com/你的用户名/email-client-py.git
cd email-client-py
```

```bash
pip install -r requirements.txt
```

## 用户快速开始

### 1. 下载APK

从 [Releases](https://github.com/你的用户名/email-client-py/releases) 页面下载最新版本的APK文件。

### 2. 安装APK

1. 将APK传输到Android设备
2. 打开APK文件
3. 允许"安装未知来源应用"
4. 点击安装

### 3. 配置邮箱

#### QQ邮箱

1. 打开应用，选择"QQ邮箱"
2. 输入QQ邮箱地址（如：123456789@qq.com）
3. 输入授权码（不是QQ密码）
4. 点击登录

**获取QQ邮箱授权码：**
1. 登录 https://mail.qq.com
2. 设置 -> 账户
3. 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
4. 开启"IMAP/SMTP服务"
5. 生成授权码
6. 使用授权码登录应用

#### 163邮箱

1. 打开应用，选择"163邮箱"
2. 输入163邮箱地址
3. 输入授权码
4. 点击登录

**获取163邮箱授权码：**
1. 登录 https://mail.163.com
2. 设置 -> POP3/SMTP/IMAP
3. 开启"IMAP/SMTP服务"
4. 设置授权码
5. 使用授权码登录应用

#### Gmail

1. 打开应用，选择"Gmail"
2. 输入Gmail地址
3. 输入应用专用密码
4. 点击登录

**获取Gmail应用专用密码：**
1. 访问 https://myaccount.google.com/security
2. 开启两步验证
3. 生成应用专用密码
4. 使用该密码登录应用

### 4. 使用功能

#### 查看邮件
- 登录后自动显示收件箱
- 点击邮件查看详情
- 点击"刷新"更新邮件列表

#### 发送邮件
- 点击"写邮件"按钮
- 填写收件人、主题、正文
- 点击"发送"

#### 回复邮件
- 在邮件详情页点击"回复"
- 编辑回复内容
- 点击"发送"

## 常见问题

### Q: 为什么需要授权码而不是密码？

A: 为了安全，大部分邮箱服务商要求第三方应用使用授权码而非登录密码。授权码可以随时撤销，不会影响邮箱账户安全。

### Q: 登录失败怎么办？

A: 检查以下几点：
1. 是否使用了授权码（不是登录密码）
2. 是否开启了IMAP/SMTP服务
3. 网络连接是否正常
4. 服务器地址和端口是否正确

### Q: 可以添加多个邮箱吗？

A: 当前版本暂不支持多账户切换，后续版本会添加此功能。

### Q: 邮件数据存储在哪里？

A: 所有数据（包括密码）都加密存储在手机本地，不会上传到任何服务器。

### Q: 卸载应用后数据会丢失吗？

A: 是的，卸载应用会清除所有本地数据，包括账户信息和缓存的邮件。
